# ISO (International Organization for Standardization)

## What it is

- International standards body; a federation of national standards bodies, one per country
- National members include ANSI (USA), JISC/JSA (Japan), DIN (Germany), BSI (UK), and around 168 others
- Headquartered in Geneva, Switzerland
- Non-governmental; not a UN body, though it collaborates with UN agencies

## History

- Founded in 1947 in Geneva
- Successor to the ISA (International Federation of the National Standardizing Associations), which operated 1926–1942

## Character Sets and Encodings

- **ISO 646** (1967) — international equivalent of ASCII; defines a 7-bit character set for the Latin alphabet, with some characters varying by country (JIS X 0201 is Japan's national variant)
- **ISO 8859** family (1980s) — 8-bit extensions of ISO 646 for various languages and scripts (e.g. ISO 8859-1 for Western European languages)
- **ISO 10646** — the Universal Character Set (UCS); ISO's counterpart to Unicode, kept in sync with the Unicode Standard

### Relation with Unicode

- ISO 10646 and Unicode are maintained in sync — same character repertoire, effectively the same standard from two different bodies
- ISO 10646 provides formal intergovernmental recognition that the Unicode Consortium (a private US consortium) cannot
